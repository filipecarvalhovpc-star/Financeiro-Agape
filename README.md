# Sistema Financeiro Ágape

## Visão Geral

Sistema financeiro desenvolvido para administração da Igreja AD Ágape e das finanças pessoais do Pr. Filipe Carvalho.

Este projeto substituirá definitivamente as planilhas financeiras utilizadas atualmente.

---

## Objetivos

- Simples
- Seguro
- Rápido
- Sincronizado
- Offline
- Responsivo
- Instalável (PWA)
- Compatível com iPhone
- Preparado para evolução

---

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Google Apps Script
- Google Drive
- GitHub Pages
